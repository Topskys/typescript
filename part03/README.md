# 面向对象
> 面向对象，即面向对像，一切皆对象，通过对象完成我们的需求。对象，包括属性和方法，而类就是对象的模型，定义对象的特征和行为。
扩展
面向对象的特征：对象唯一性、抽象性、继承性、多态性。
UML(Unified Modeling Language)，统一建模语言，将是面向对象技术领域内占主导地位的标准建模语言。
